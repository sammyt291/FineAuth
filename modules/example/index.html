Module Example: Visuals, Text, & Code

This module demonstrates a complete package: a visual hero section, supporting text blocks, and code snippets that show how to structure a module in FineAuth.

Visual Pattern

Use gradients, cards, and spacing helpers to match the default FineAuth aesthetic. Keep contrast high and corners rounded for consistency.

Text Guidance

Provide concise instructions so members know what to do next. Link back to built-in modules, like Home or Navigation.

Interactive Example

Modules can send users to the parent shell using window.parent.location.hash. Try the buttons above to jump to another module.

Code Example

Minimal module manifest + iframe-friendly markup:

{
  "name": "my-module",
  "displayName": "My Module",
  "description": "Custom module example",
  "mainPage": "index.html"
}

<script>
  document.querySelector('[data-action="home"]').addEventListener('click', () => {
    window.parent.location.hash = '#/module/home';
  });
</script>
